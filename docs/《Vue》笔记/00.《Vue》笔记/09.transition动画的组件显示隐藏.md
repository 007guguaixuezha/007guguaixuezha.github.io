# transition动画的组件显示隐藏

```html
<template>
    <transition name="slide">
         <div class="add-song">
             ...
        </div>
    </transition>    
<template>
```

```css
 .add-song
    position: fixed
    top: 0
    bottom: 0
    width: 100%
    z-index: 200 
    &.slide-enter-active, &.slide-leave-active
      transition: all 0.3s
    &.slide-enter, &.slide-leave-to
      transform: translate3d(100%, 0, 0)
```
