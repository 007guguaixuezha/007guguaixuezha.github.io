# 父组件给子组件传值



```js
父组件：
<子组件标签 v-bind:值名称="值"></子组件标签 >

子组件：
 props: {
    值名称: { // 接收父组件传入值
        type: String,
        default: ''
    }
 }
```



**Q：vue父组件怎么给子组件传值？**

A：父组件中，在子组件标签使用v-bind绑定值，在子组件内通过props对象接收传入的值。
