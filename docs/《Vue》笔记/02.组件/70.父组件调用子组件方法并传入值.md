# 父组件调用子组件方法并传入值

```js
父组件：
<子组件标签  ref="searchBox"></子组件标签>

methods: {
    addQuery(query) {
      this.$refs.searchBox.setQuery(query) // 调用子组件方法并传入值
    }
}

子组件：
methods: {
    setQuery(query) {
      this.query = query
    }
  }
}

```
